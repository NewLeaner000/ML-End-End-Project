## End to ENd ML Project
Welcome to my end-to-end machine learning project. 

The main goal of this repo is to build out a complete machine learning pipeline—from raw data and model training, all the way to a working deployment. It's built to be simple, readable, and easy to run.

To serve the actual predictions, I'm using **Flask** to handle the model inferences. The Flask app runs a simple web interface that takes in your data, feeds it to the trained model in the backend, and spits out the result.
